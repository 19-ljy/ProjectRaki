# 必选-遗传学

## 什么是必选课

必选课与选修课不同，必选课是在限定的几门课中，至多有2门课可以不休。（多修这两门算D类选修学分）

伯苓班、生物科学是8选6，生物技术是9选7.

属于前者的必选课：植物学、动物学、植物生理学、动物生理学、细胞生物学、微生物学、分子生物学、遗传学。**(均含实验课)**

属于后者的必选课：植物学、动物学、细胞生物学、遗传学、微生物学 **（此五门含实验课）** ；微生物发酵工程、开放实验专题I、基因操作原理、分子生物学实验 **（无理论课）** 。

--W.M. 2021
